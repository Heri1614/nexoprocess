Sistema de Acciones Gemba

Microsoft Forms SharePoint Excel Automatizado
[ Espacio reservado para incrustar el Video Explicativo o la Imagen del Dashboard ]

El Problema

El levantamiento de acciones operativas en piso (Gemba) dependía de anotaciones manuales, lo que generaba retrasos en la captura de datos, pérdida de información y una nula visibilidad en tiempo real para la toma de decisiones.

La Solución

Se desarrolló un flujo digital donde los operadores ingresan datos vía web usando Microsoft Forms. Esta información alimenta automáticamente una base de datos centralizada en SharePoint, eliminando la transcripción manual.

El Resultado

Un archivo de Excel local se conecta a SharePoint y se actualiza al instante, mostrando un Dashboard dinámico. Esto redujo el tiempo de procesamiento de datos a cero y mejoró el seguimiento de las acciones operativas.
